# 网页图片一键下载 - Chrome扩展

一键下载网页所有图片并生成PDF的Chrome浏览器扩展。

## 📋 扩展信息

- **扩展名称**: 网页图片一键下载
- **版本**: 1.0.0
- **作者**: Zjmainstay
- **类型**: Chrome浏览器扩展

## ✨ 功能特点

- 🔢 **顺序保证**: 严格保持图片在PDF中的顺序与网页一致
- 📄 **一键下载**: 2步完成操作（点击图标 → 点击下载）
- 🎯 **智能过滤**: 自动过滤小图标（可自定义最小尺寸）
- 📊 **实时进度**: 显示加载和生成PDF的进度
- 🎨 **界面友好**: Google Material Design风格

## 🚀 使用方法

1. 安装扩展到Chrome浏览器
2. 打开包含图片的网页（如在线电子书、图库）
3. 点击浏览器工具栏的扩展图标
4. 查看检测到的图片预览
5. 点击"下载PDF"按钮
6. 选择保存位置，完成下载
